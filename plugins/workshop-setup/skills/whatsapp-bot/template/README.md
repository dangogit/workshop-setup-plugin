# WhatsApp Claude Agent

סוכן מקומי שמחבר בין WhatsApp ל-Claude Code. הממשק מסביר איפה הסוכן מקשיב, מי רשאי לדבר איתו ומה מצב החיבור.

## הפעלה

- Mac: לחיצה כפולה על `start.command`
- Windows: לחיצה כפולה על `start.bat`
- Linux: `bash start.command`

הממשק נפתח בכתובת `http://127.0.0.1:7654`.

דרישות: Node.js 20.9 ומעלה, ו-Claude Code CLI.

## הגדרה ראשונה

1. סורקים QR דרך WhatsApp, הגדרות, מכשירים מקושרים, קישור מכשיר.
2. בוחרים דרך עבודה:
   - מספר אחד: שיחה עם הסוכן בצ'אט עם עצמך או בקבוצה פרטית שבחרת. לא צריך מספר נוסף.
   - מספר נפרד לבוט: המספר המחובר הוא הבוט. שולחים קוד חיבור מהמספר האישי בהודעה פרטית.
   - בוט לקבוצה או ללקוחות: בוחרים קבוצה ספציפית. משתתפים מפעילים אותו בתיוג, עם הרשאות מוגבלות.
3. שולחים הודעת בדיקה בצ'אט שנבחר.

הכרטיס "מקשיב" במסך הראשי מציג בדיוק איפה הסוכן אמור לענות.

## פקודות WhatsApp

| פקודה | פעולה |
|---|---|
| `/help` | הצגת הפקודות |
| `/reset` | איפוס זיכרון השיחה הנוכחית |
| `/cancel` | עצירת המשימה הפעילה בשיחה הנוכחית |
| `/pwd` | הצגת תיקיית העבודה הפעילה |
| `/cd ~/Projects/myapp` | שינוי תיקיית פתיחה בשיחה, במצב עוזר אישי |
| `/model opus` | החלפת מודל |
| `/session` | מידע על זיכרון השיחה |

## הרשאות ותיקיית עבודה

תיקיית העבודה היא נקודת הפתיחה של Claude, לא גבול אבטחה. במצב עוזר אישי עם הרשאות מלאות, Claude יכול לקרוא ולשנות קבצים אחרים במחשב ולהריץ פקודות. יש לאשר רק מספרים שסומכים עליהם.

מצב קבוצה או לקוחות משתמש בהרשאות מוגבלות, בקבוצות שנבחרו בלבד.

## פרטיות

הסוכן שומר במחשב:

- חיבור WhatsApp בתיקיית `auth/`
- עד 60 הודעות אחרונות ב-`feed.json`
- זיכרון שיחות ב-`sessions.json`
- מדיה שהתקבלה בתיקיית `media/`
- לוג טכני והגדרות, כולל מפתח OpenAI אם הוגדר

בממשק, הגדרות, פרטיות, אפשר למחוק את כל הנתונים המקומיים. המחיקה מנתקת את WhatsApp ומחזירה את הסוכן להגדרה ראשונה.

הודעות עוברות דרך WhatsApp ו-Claude. הודעות קוליות עוברות גם דרך OpenAI אם הפעלת תמלול או קול.

## בדיקת מוכנות

במסך הראשי מופיעה בדיקה ל-Node.js, Claude Code, תקינות ההגדרות, חיבור WhatsApp והרשאת כתיבה מקומית.

אפשר לבדוק גם מהטרמינל:

```bash
curl -fsS http://127.0.0.1:7654/doctor
```

## פתרון תקלות

- הממשק לא נפתח: פתח ידנית `http://127.0.0.1:7654` ובדוק את חלון ההפעלה.
- פורט 7654 תפוס: המפעיל לא עוצר תוכנות אחרות. זהה את התהליך או בחר פורט אחר.
- Claude לא מוכן: הרץ `claude --version`.
- QR לא עובד: לחץ "סריקה מחדש" וסרוק QR חדש.
- אין תשובה במספר אחד: בדוק שכרטיס "מקשיב" מציג צ'אט עם עצמך או את הקבוצה הנכונה.
- אין תשובה בקבוצה: ודא שהקבוצה מופעלת ושבוצע תיוג במצב תיוג.
- קוד חיבור לא נקלט: שלח רק את שש הספרות בהודעה פרטית מהמספר האישי.

## התקנה ידנית של רכיבים

```bash
npm install --ignore-scripts --no-fund --no-audit
```

`--ignore-scripts` הוא חלק קבוע מהתקנת הסדנה.

## הערת WhatsApp

החיבור משתמש ב-Baileys, ספריית WhatsApp Web לא רשמית. שינוי בצד WhatsApp עלול לדרוש סריקה מחדש או עדכון. לשירות לקוחות בייצור מומלץ להשתמש ב-WhatsApp Business Platform הרשמי.
